## Features

- 2FA
- ACL
- Anti Phishing Code on email
- Audit
- CORS
- Device authorization
- Etag
- Horizon
- Laravel 5.7
- Login
- Login history
- Multiple localizations
- Password reset
- Password must not be in one of the 4 million weak passwords
- PHPCS PSR2
- Register
- Swoole
- Tests
- uuid

Soon:

- Job example
- Schedule example
